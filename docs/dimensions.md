# 📐 Hand Dimension Data

This document contains the measurement data used as the dimensional reference for the Bionic Hand CAD model. All measurements are self-measured from the designer's hand.

---

## Measurement Methodology

- **Tool used:** Ruler + digital caliper (where available)
- **Reference point:** All lengths measured along the dorsal (back) surface of the hand
- **Joint widths:** Measured at the widest point of each joint
